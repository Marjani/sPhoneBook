The idea behind of this project is that show how to work with Ajax in ASP.NET without using UpdatePanel .

In this project I use jQuery Ajax Methods for transfering data between client and server, and use SP for working in Database.

Requirement
--------------------------------------
- Sql Server 
- IIS 
- Use VS2010 for run and debug



Installation
--------------------------------------
Only run Person.sql command in a database and change the connection string in web.config file.
Open project in VS2010 or create a website in IIS and put this files in your website.
